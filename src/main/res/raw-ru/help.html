Справка на программу «Шкет»

Создание ярлыков для Intent Radio и VLC.

Создание ярлыков

Отображаются поля для ввода Имя и Url радиостанции. Заполните поля и нажмите кнопку Создать чтобы создать ярлык. Кнопка Отмена закроет отображение. Кнопка Список на панели инструментов откроет отображение списка.

VLC-проигрыватель

Программа «Шкет» будет создавать ярлыки и воспроизводить станции только в VLC. Варианты Стоп, Пауза и Продолжить не принимаются. Управляйте воспроизведением через уведомления VLC.

Меню

  • Тёмная тема – Только в тёмном режиме.
  • Через VLC – Только в VLC-проигрывателе.

Искать станции

Отображение поиска показывает те же поля, которые можно заполнить, выбрав элемент в списке или через клавиатуру. Кнопка Добавить добавит запись. Кнопка Удалить удалит выбранную запись.

Поиск

Коснитесь значка Поиск на панели инструментов, чтобы начать интерактивный поиск в списке станций. Затем коснитесь значка Поиск на клавиатуре, чтобы выбрать первый элемент из списка, или коснитесь элемента в списке.

Воспроизвести станции

Кнопка Выбрать закроет список и вернется в исходное положение. Используйте кнопки панели инструментов может Воспроизвести, Стоп, Паузы или Продолжить выбранной станции на Intent Radio напрямую.

Сохранить и Восстановить

Для сохранения и восстановления текущего списка станций можно использовать пункты в меню Сохранить и Восстановить. Расположение списка тут Shorty/entries.json. Доступ к этой папке возможен из Проводника или с подключенного ПК. Список в JSON формате.

Импортировать

Пункт в меню Импортировать запрашивает файл, по умолчанию extras.csv в том же месте. Вы можете изменить название и/или расположение. Файл должен быть в CSV формате, который можно создать и сохранить из эл.таблицы данных. Если записи найдены, то они будут добавлены в список. Функция не будет добавлять дубликаты записей, которые уже есть в списке, а добавит дубликаты в сам файл.

Дополнительная информация

Если программа «Шкет» удалена, то ярлыки, использующие его в фоновом режиме, перестанут работать. Не создаст ярлыки, если не установлена программа Intent Radio или VLC.
